programación funcional Python

La programación funcional es un paradigma de programación que trata a las computaciones como evaluación de funciones matemáticas y evita cambiar estados y datos mutables. En Python, un lenguaje de programación multiparadigma, también puedes escribir código utilizando enfoques funcionales.

Funciones como objeto de primero orden

las funciones son de primera clase, lo que significa que pueden ser tratadas como cualquier otro objeto, como enteros, cadenas o listas. Esto permite que las funciones se pasen como argumentos a otras funciones, se devuelvan como valores desde funciones, se asignen a variables y se almacenen en estructuras de datos

Ejemplo:
def aplicar_operacion(func, x): return func(x) def cuadrado(n): return n * n def doble(n): return n * 2 resultado_cuadrado = aplicar_operacion(cuadrado, 5) resultado_doble = aplicar_operacion(doble, 3) print(resultado_cuadrado) # Salida: 25 print(resultado_doble) # Salida: 6

Recursión sobre iteración

La recursión y la iteración son dos enfoques distintos para la resolución de problemas en programación. Ambos tienen sus ventajas y desventajas, y la elección entre ellos a menudo depende del problema específico y de la preferencia del programador

Ejemplo:
Recursion:
def factorial_recursivo(n): # Caso base: factorial de 0 es 1 if n == 0: return 1 # Caso recursivo: n! = n * (n-1)! else: return n * factorial_recursivo(n - 1) # Ejemplo de uso resultado_recursivo = factorial_recursivo(5) print(f"Factorial (recursivo) de 5: {resultado_recursivo}")
iteracion:
def factorial_iterativo(n): resultado = 1 # Iteración desde 1 hasta n for i in range(1, n + 1): resultado *= i return resultado # Ejemplo de uso resultado_iterativo = factorial_iterativo(5) print(f"Factorial (iterativo) de 5: {resultado_iterativo}

Procesamiento de listas

El procesamiento de listas es una parte esencial de la programación en Python, y hay varias formas de trabajar con listas

Ejemplo:
Recorrer una lista:
mi_lista = [1, 2, 3, 4, 5] for elemento in mi_lista: print(elemento)
Concatenación de Listas:
lista1 = [1, 2, 3] lista2 = [4, 5, 6] concatenada = lista1 + lista2 print(concatenada) # Salida: [1, 2, 3, 4, 5, 6]
Eliminar Elementos Duplicados
duplicada = [1, 2, 2, 3, 4, 4, 5] sin_duplicados = list(set(duplicada)) print(sin_duplicados) # Salida: [1, 2, 3, 4, 5]

Funciones puras

Las funciones puras no tienen efectos secundarios y siempre producen el mismo resultado dado el mismo conjunto de entradas. Evitan modificar estados globales o datos fuera de la función

Ejemplo:
Función Pura:
# Función pura def suma(a, b): return a + b # Ejemplos de uso resultado1 = suma(3, 5) resultado2 = suma(3, 5) print(resultado1) # Salida: 8 print(resultado2) # Salida: 8
La función suma es pura porque, dado el mismo conjunto de entradas (a y b), siempre producirá el mismo resultado sin efectos secundarios.

Funciones de alto orden

puedes pasar funciones como argumentos a otras funciones y devolver funciones desde funciones. Esto permite construir funciones de orden superior, que son funciones que operan sobre otras funciones

* puede recibir como parámetro otra funciones * pueden ser pasadas como parámetro * Map
* Filter
* Reduce

Ejemplo: # Ejemplo de map(), filter() y reduce() como funciones de alto orden from functools import reduce # Función para elevar al cuadrado def cuadrado(x): return x ** 2 # Función para filtrar números pares def es_par(x): return x % 2 == 0 # Función para sumar elementos de una lista def suma(x, y): return x + y # Lista de números numeros = [1, 2, 3, 4, 5] # Usando map() para aplicar cuadrado a cada elemento cuadrados = list(map(cuadrado, numeros)) # Usando filter() para obtener números pares pares = list(filter(es_par, numeros)) # Usando reduce() para sumar los elementos de la lista suma_total = reduce(suma, numeros) print(cuadrados) # [1, 4, 9, 16, 25] print(pares) # [2, 4] print(suma_total) # 15

Funciones lambda

Las expresiones lambda permiten crear funciones anónimas y se utilizan comúnmente en programación funcional.

Ejemplo:
Función lambda:
# Sintaxis básica de una función lambda suma = lambda x, y: x + y # Uso de la función lambda resultado = suma(3, 5) print(resultado) # Salida: 8
En este ejemplo, lambda x, y: x + y es una función lambda que toma dos argumentos (x e y) y devuelve su suma. La función lambda se asigna a la variable suma, y luego puedes usar esa variable como si fuera una función normal.
